Manim

先进入虚拟环境，然后 pip install manim 即可

!!! note
如果是用上面的方法编译安装的 Python，可能会报缺少 _bz2，原因是 bzip2 是 Python 编译的可选依赖，编译时没有带上

如果缺少 _bz2，执行 sudo apt install libbz2-dev 然后重新编译安装 Python
